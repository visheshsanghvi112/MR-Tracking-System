IF YOU DON'T SEE THIS
================================================================

Problem 1: No [DEBUG] lines
---------------------------
Backend is using old smart_sheets.py without debug
Solution: Check if smart_sheets.py was saved properly

Problem 2: Startup test shows 0 records
---------------------------------------
Sheets connection works but date/MR_ID doesn't match
Solution: Check Date column format in your sheets

Problem 3: No startup test at all
----------------------------------
Backend failed to load
Solution: Check console for errors

================================================================
MANUAL RESTART STEPS
================================================================

1. Find backend terminal (shows "Uvicorn running")
2. Click on it to focus
3. Press Ctrl+C
4. Wait 2 seconds
5. Type: python api/main.py
6. Press Enter
7. WATCH THE OUTPUT - look for:
   - "Reloading smart_sheets module..." 
   - "TEST PASSED: Found 1 records"
   - "[DEBUG] Parsed from Remarks: lat=18.947934"

8. If you see all 3: SUCCESS! Backend loaded new code
9. If you don't see them: Something's wrong
